Environmental Setup.
-

1.Download and Install the NodeJS

	Website : https://nodejs.org/en/download/
	file name : node-v6.10.3-x64.msi

2. Download and Install the GIT

	Website : https://git-scm.com/download/win
	file name : Git-2.13.0-64-bit.exe


3.Download WebStrom IDE.

	website : https://www.jetbrains.com/webstorm/download/download-thanks.html?platform=windows

	file name : 	Webstrom-2017.1.3.exe


4. Create the Account with GitHub.

	Website : https://github.com/join


5. Checkout the Code From GitHub.

	URL : https://github.com/ajsnaresh/AJS_Classes.git


6.Download and install the MongoDB.

    Website : https://www.mongodb.com/dr/fastdl.mongodb.org/win32/
              mongodb-win32-x86_64-2008plus-ssl-3.4.4-signed.msi/download

    file name : mongodb-win32-x86_64-2008plus-ssl-3.4.4-signed.msi


    Set the path in Environmental Setup.


    Create the Following Folder Structure in Installation Drive.

    C:\data\db.


7. Install the bower

    > npm install -g bower
