Create the Repository Locally
-----------------------------
	
	- create the directory (folder)
		
	Ex.
		github demo


	- launch the "gitbash here" terminal on
          rigth click of directory

	- execute the following command

	> git init

[Note : Automatically "github demo" folder converted
        as local repository]

Add the User Credentials to the Repository
------------------------------------------
1. name
>git config --global user.name "NareshIT"


2. email
>git config --global user.email "hr@nareshit.in"


Create the Files to the Repository
----------------------------------

> touch sample.html

[Note : "touch"	command automatically creates
        the files in repository]

[Note : by default the files are in unsaved /  
        uncommitted / untracked mode]

> git status
	- used to know the status of files

[Note : "red color labeled" files indicating the
        untracked files in repository]
	

Commit the Files to the Repository
----------------------------------
> git add sample.html

> git commit

> press "i" [Enabling the --INSERT-- mode]

> type "initial commit"

> press "ESC"

> type ":wq"

> press "Enter"


from 2nd commit onwards:
------------------------
> git add sample.html

> git commit -m "let keyword introduced"

> git log

	- used to know the number of commits
          in repository. 


Create the Branches
-------------------

> git branch AngularJS_3PM

> git checkout AngularJS_3PM

[Note : automatically child branches contains
        the master branch files]
Merge the Branches
------------------

> git checkout master

> git merge AngularJS_3PM


Stash Changes / Unstash Changes In Repository
---------------------------------------------

   - Holding the changes tempararly called as
     Stash.

  -  UnHolding the temparary changes called as	
     unStash

> git stash(0)
	- making the code as hold in 0th index.

> git stash pop(0)
	- making the 0th index code as unhold.
